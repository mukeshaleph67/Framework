# PGColor

```
Platform : iOS
version : 0.0.1
```

## Usage

`PGColor.color` contains our color palette for uob design system. follow the steps below to use the provided colors.

```
import Playground
apply the color for example shown as below for cell background color
cell.backgroundColor = UIColor.color(name: PGColorType.mochiLight)

```
